# **App Name**: CreatorX SEO

## Core Features:

- Content Idea Generation: Generate content ideas based on category, language, and format using a generative AI tool; includes SEO title variations, viral hooks, thumbnail concepts, short descriptions, tags, structure points, and repurpose suggestions.
- Keyword Research: Provide search volume, trend data, difficulty score, long-tail variants, keyword clusters, and potential views using data aggregation tools and AI to determine trends.
- Competitor Analysis: Analyze competitor channels/videos for top content, watch time, headline patterns, tags, and content gaps; suggest content angles to outrank competitors using AI analysis.
- Thumbnail Analysis: Evaluate thumbnails for readability, prominence, contrast, and clickbait using AI; predict CTR and suggest improvements, including A/B mockups.
- Rank Tracking: Track keyword rankings, display rank history, impressions → CTR → views funnel; send alerts for significant rank drops using external APIs.
- Reel Optimization: Generate viral hashtags, provide a Reel SEO score, find trending audio, and generate captions with CTAs; optimize for Instagram Reels.
- Content Repurposing & Scheduling: Convert long videos into short scripts, auto-populate a content calendar, and schedule content to connected platforms, all with user-friendly drag-and-drop interface elements.

## Style Guidelines:

- Primary color: A saturated blue (#29ABE2), providing a sense of trust and competence appropriate for an SEO application.
- Background color: A light desaturated blue (#D8EDF5), which creates a calm and professional environment.
- Accent color: A vibrant green (#90EE90), providing positive feedback to users with the sense of measurable growth they expect from an SEO tool.
- Body and headline font: 'Inter', a sans-serif, to maintain a clean, modern, neutral, objective appearance.
- Use consistent and professional icons related to SEO, analytics, and content creation.
- Maintain a clean and intuitive layout with clear sections and visual hierarchy.
- Subtle animations and transitions to enhance user experience.
